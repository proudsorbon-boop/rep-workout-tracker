Project context:
This is a minimalist fitness web app called "Fit Minimalist".
Tech stack:

- Mobile wrapper: Capacitor (Android)

- Display progress using simple charts (no heavy libraries)

3. Workout Planner
- Weekly workout schedule
- Levels: Beginner / Intermediate / Advanced

5. Notifications (logic only)

- Display badges visually

7. Nutrition / Calories (basic)

- Input: age, weight, height, activity level
- Output: daily calorie recommendation
- Optional: simple food log (text only)

8. Social / Motivation
- Daily motivational quote (random)
- Share progress summary (text based)
- Prepare UI for future social features

9. UX / Onboarding
- First-launch onboarding screens
